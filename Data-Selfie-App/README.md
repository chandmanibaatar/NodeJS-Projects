# Data Selfie App

![](zurag/fg0ddd.png)


## Setup

In your terminal:
```sh
npm init
```

```sh
npm install
```

then start your server
```sh
npm start
```
or if in development: `npm run dev`

Go to: `localhost:3030` to see the magic.
